#   おおまかな流れ

1.  最終到達目標を設定する．
2.  その日その日のToDoを作成し，項目ごとに大別を行う．
3. 大別を行ったあと，数値化して，取り組みやすいものほど数値を小さくし， すぐにとりかかれるように仕向ける．
